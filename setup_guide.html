Smart Ranch Setup & Android Transformation Guide

This document provides the step-by-step process for setting up the Smart Ranch management system on a new laptop from a cloned GitHub repository.

1. Prerequisites

Ensure these tools are installed BEFORE you begin:

Software Purpose Source
Git Cloning the source code git-scm.com
Python 3.9+ Backend API execution python.org
PostgreSQL Database storage postgresql.org
Flutter SDK Mobile app building flutter.dev
Android Studio Emulator and SDK tools developer.android.com

2. Backend Configuration

  1. Clone & Navigate: 
    git clone [your-repository-url]
    cd "smart ranch system/backend"
  2. Setup Environment: 
    python -m venv venv
    .\venv\Scripts\activate
  3. Install Logic: 
    pip install fastapi uvicorn sqlalchemy asyncpg pydantic python-dotenv "python-jose[cryptography]" "passlib[bcrypt]" bcrypt
  4. Configure Database:

    Create a .env file in the backend folder:

    DATABASE_URL=postgresql+asyncpg://[user]:[password]@localhost:5432/smartranch
  5. Launch Server: 
    uvicorn main:app --reload --host 0.0.0.0 --port 8000
NOTE: Ensure your PostgreSQL service is running and you have created a database named 'smartranch'.

3. Mobile Application Setup

  1. Install Dependencies: 
    cd ../mobile_app
    flutter pub get
  2. Set Endpoint:

    Verify lib/services/api_service.dart:

    static String get baseUrl => 'http://10.0.2.2:8000';
  3. Run on Android:

    Start your emulator in Android Studio, then run:

    flutter run
TIP: If you face "license not accepted" errors, run flutter doctor --android-licenses.
IMPORTANT: Always ensure the backend is running BEFORE launching the mobile app to avoid connection timeout errors.
