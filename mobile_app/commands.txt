Smart Ranch Management System - Walkthrough
Prerequisites

PostgreSQL: Must be installed and running.
Flutter SDK: Must be installed and configured.
Python 3.8+: Must be installed.

1. Database Setup
Ensure PostgreSQL is running.
Create a database named smartranch (or update 
backend/database.py
 with your DB name).

Run the initialization script:

psql -U postgres -d smartranch -f init.sql
Note: The backend will also attempt to create tables on startup if they don't exist.
2. Backend Setup (FastAPI)
Navigate to the backend directory:
cd backend
Create a virtual environment and install dependencies:
python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
Configure Database:
Open 
database.py
.
Update DATABASE_URL if your credentials differ from postgres:password@localhost/smartranch.
Run the server:
uvicorn main:app --reload
The API will be available at http://127.0.0.1:8000.
Swagger UI docs: http://127.0.0.1:8000/docs.

3. Frontend Setup (Flutter)
Navigate to the mobile_app directory:
cd mobile_app
Install dependencies:
flutter pub get
Configure API URL:
Open 
lib/services/api_service.dart
.
If running on an Android Emulator, use http://10.0.2.2:8000.
If running on Windows/Web, use http://127.0.0.1:8000.
Run the app:
flutter run

4. Verification
You can run the included test script to verify the backend API:
cd backend
python test_api.py
to run the app:
python -m uvicorn main:app --reload --host 127.0.0.1 --port 8000

to run the app in production:
uvicorn main:app --reload
uvicorn main:app --reload --host 0.0.0.0 --port 8000

To run it on Android from your Windows machine, ensure you meet the following setup requirements:

Install Android Setup Tools: If you haven't yet, install Android Studio. During installation, make sure to include the Android SDK, Android SDK Command-line Tools, and Android Emulator.

Accept Android Licenses: Open a terminal and run the following command to finalize your SDK setup: flutter doctor --android-licenses (Press 'y' to accept the prompts)

Start an Android Emulator (or Plug in a Device):

Emulator: Open Android Studio -> Device Manager -> Create a Virtual Device -> Select a Phone (e.g., Pixel 6) -> Start it.
Physical Device: Connect an Android phone to your Windows PC via USB. 
Ensure USB Debugging is turned on inside your phone's Developer Options.
Run the App: Once your emulator is running or your phone is plugged in, navigate your terminal to the root app folder: cd c:\dev\smart-ranch\mobile_app Then execute the build and install command: flutter run -d android

This will compile the Kotlin/Java files, bundle your Dart code, and automatically open the identical Smart Ranch app you've been working on, natively on your Android system! Let me know if you run into any SDK/Gradle build errors while executing step 4.
